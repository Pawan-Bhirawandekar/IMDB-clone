🎬 IMDb Clone (React JS)

A frontend-focused IMDb Clone built using React JS. Users can browse movies, view ratings, and manage a personal watchlist.

🚀 Features

  🔍 Search Movies
  
  ⭐ View Ratings & Popularity
  
  🗂️ Display Movie Genres
  
  📌 Add / Remove from Watchlist
  
  🔼 Sort by Rating
  
  💾 Local Storage Persistence
  
  📱 Responsive UI

🛠️ Tech Stack

**React JS**
  
  Tailwind CSS
  
  TMDB API
  
  React Hooks
  
  Lucide Icons

📦 **Installation**
git clone https://github.com/your-username/imdb-clone.git
cd imdb-clone
npm install
npm start
